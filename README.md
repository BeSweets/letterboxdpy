## Installation

```
    pip install letterboxdpy
```

## **Code Example**

```python
from letterboxdpy import user
nick = user.User("nmcassa")
print(nick.jsonify())
```

## **Code Example**

```python
from letterboxdpy import movie
king = movie.Movie("king kong")
print(king.jsonify())

king = movie.Movie("king kong", 2005)
print(king.jsonify())
```
